*Bunyi Suara Musik*

: bang, berenti bang berenti.

*Suara Musik Diam*

: bang, kira-kiralah bang, emang kaga liat apa, disini lagi ada ulang tahun desa?
: ya tau lah bang.
: udah tau kenapa masih berisik disini.
: makanya saya mau kasih tau sama abang. diatas meja ada plastik, isinya tahu ama tempe.
: maksud abang
: emang sengaja kami brisik, biar pada tau tuan muda tanjung bulan udah sampe.

*sorak*

: waduuh, jadi ini rombongan tuan muda tanjung bulan?
: iya bang
: bang, kalo mau ke balai desa, jangan bawa bantal guling.
: maksud abang
: kata abang ini rombongan tuan muda, nape itu ada yg mirip kuda lumping?

*sorak*

: bang, gapapa kalo cuma bawa bantal guling, asal kaga tidur di aspal.
: maksud abang
: emang sengaja dimirip-miripin kuda lumping, biar berasa kearifan lokal.

*sorak*

: bang, kalo bingung boleh nanya, tapi jangan basa basi.
: maksud abang
: saya kagak peduli ni rombongan apa namanya,yang jelas kalo lewat kemari kudu permisi.
: bang, jalan-jalan kepasar jumat, pulangnya beli ikan teri.
: maksud abang
: kami rombongan tanjung bulan dengan segala hormat, tolong diterima dengan senang hati.

*sorak*

: waduh main terima aja, perkara terima mah urusan gampang bang.
: terus
: cuma begini bang. kalo abang makan ikan teri, jangan dibuang kepalanya.
: maksud abang
: kalo rombongan tanjung bulan udah niat dateng kemari, saya pengen tau apa maksud dan tujuannya.
: bang, jangan joget di jalan raya, apalagi sampe dansa
: maksud abang
: dah pasti kami mau ikut karnaval budaya,untuk memeriahkan ulang tahun desa!
